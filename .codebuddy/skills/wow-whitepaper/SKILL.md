# PUBGM WoW 版本白皮书功能陈述撰写

> 将简短功能关键词扩写为标准的版本白皮书功能点陈述句。

## 概述

本 Skill 用于协助整理 PUBGM WoW 模式（UGC 玩法）的版本白皮书功能列表。输入简短的功能关键词或短语，输出标准、专业且信息量完整的版本功能陈述。

## 命令

| 指令 | 功能 | 参数 | 示例 |
|------|------|------|------|
| `/baipishu` | 扩写功能陈述 | `<关键词列表>` | `/baipishu 动画生成, 自定义UI, 全局变量` |
| `/baipishu --version <版本号>` | 指定版本号扩写 | 版本号 + 关键词 | `/baipishu --version 4.4.0 动画生成` |
| `/baipishu --category <分类>` | 按分类扩写 | 分类名 + 关键词 | `/baipishu --category 可视化编程 循环节点` |

## 扩写规则

### 核心句式（严格遵循）

每条功能陈述**必须**套用以下句式结构：

```
[序号]. 新增[功能名称]功能，支持[具体机制/操作方式]，用于[应用场景/关联的设备或系统]。
```

**句式变体**（当功能偏向优化/调整时）：

```
[序号]. 优化[功能名称]，调整[具体改动内容]，提升[优化目标/体验改善点]。
```

```
[序号]. 扩展[功能名称]能力，新增[新增的子功能/参数/选项]，适用于[扩展后的新场景]。
```

### 扩写思维链

对每个输入关键词，按以下步骤推理后输出：

1. **识别功能本体**：该关键词在 PUBGM WoW 中对应什么功能模块？
2. **推导操作机制**：用户/创作者通过什么方式使用该功能？（上传、拖拽、配置节点、输入参数等）
3. **关联应用场景**：该功能最终作用于什么设备、系统或玩法环节？
4. **套用句式输出**：将上述三要素填入核心句式

### 专业术语规范

使用 PUBGM WoW 模式常用术语，详见 [references/wow-terminology.md](references/wow-terminology.md)。

核心术语速查：

| 术语 | 说明 |
|------|------|
| 可视化编程 | WoW 编辑器中的蓝图/节点式编程系统 |
| 自定义 UI | 创作者自由设计的游戏内界面元素 |
| 全局变量 | 可视化编程中跨作用域共享的数据变量 |
| 互动物体 | 场景中可被玩家交互的物件（开关、按钮、机关等） |
| 武装 AI | WoW 中的可编程 NPC/敌人实体 |
| 虚拟投影装置 | 可在场景中播放动画或特效的投影设备 |
| 载具编辑器 | 用于自定义载具属性和行为的编辑工具 |
| 预制件 | 可复用的场景组合单元 |
| 资源商城 | 创作者上传和下载资源的平台 |
| 地编 | 地图编辑器 |
| 事件系统 | 可视化编程中基于触发器的事件响应机制 |

### 文体规范

1. **陈述事实**：只描述功能本身，不发表评论或评价
2. **不用感叹号**：全部使用句号结尾
3. **不用营销语**：禁止"快来试试"、"全新体验"、"震撼升级"等词汇
4. **简洁专业**：一个功能点一句话，不超过 80 字
5. **数学公式**：涉及数值逻辑时使用 LaTeX 格式，如 $y = x + 1$
6. **中文标点**：全部使用中文标点符号

## 输出格式

### 单条模式

当用户输入单个关键词时，直接输出一条功能陈述：

```
1. 新增动画生成功能，支持作者上传视频后生成对应骨骼动画，用于可视化编程控制武装AI和虚拟投影装置。
```

### 批量模式

当用户输入多个关键词（逗号分隔或换行）时，按序号连续输出：

```
1. 新增动画生成功能，支持作者上传视频后生成对应骨骼动画，用于可视化编程控制武装AI和虚拟投影装置。
2. 新增自定义UI编辑器，支持创作者自由拖拽按钮与图片布局，用于制作个性化的游戏界面与交互菜单。
3. 新增全局变量管理面板，支持跨蓝图的变量声明与读写操作，用于多模块间的数据共享和状态同步。
```

### 分类模式（--category）

当指定分类时，在输出前添加分类标题：

```markdown
### 可视化编程

1. 新增循环节点功能，支持在蓝图中配置 For/While 循环逻辑，用于批量处理数组数据和重复执行操作序列。
2. 新增函数封装节点，支持将多个节点组合封装为可复用的自定义函数，用于简化复杂蓝图逻辑和提升编程效率。
```

### 版本模式（--version）

当指定版本号时，在输出头部标注版本：

```markdown
## PUBGM WoW V4.4.0 版本功能列表

1. 新增动画生成功能，支持作者上传视频后生成对应骨骼动画，用于可视化编程控制武装AI和虚拟投影装置。
2. ...
```

```
⚠️ 关键词"{keyword}"较为宽泛，可能对应多个功能方向：
1. {可能方向 A}
2. {可能方向 B}
3. {可能方向 C}
请确认您想要扩写的具体方向，或直接输入更详细的描述。
```

## 注意事项

1. **贴合实际**：扩写内容需基于 PUBGM WoW 编辑器的真实功能逻辑，不要凭空编造不存在的机制
2. **术语一致**：同一文档内对同一概念使用统一术语，参考术语表
3. **粒度适中**：一个功能点对应一句话，不要将多个独立功能合并为一条
4. **可追溯**：扩写时如果对功能理解不确定，应向用户确认而非猜测
5. **版本对齐**：如用户指定了版本号，确保功能描述与该版本的实际规划一致
