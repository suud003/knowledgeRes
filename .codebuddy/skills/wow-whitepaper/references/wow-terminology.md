# PUBGM WoW 模式术语表

> 本文档收录 PUBGM WoW 模式（UGC 玩法编辑器）中的常用专业术语，供白皮书功能陈述扩写时参考。

## 一、编辑器核心模块

| 术语 | 英文 | 说明 |
|------|------|------|
| 可视化编程 | Visual Scripting | WoW 编辑器中的蓝图/节点式编程系统，创作者通过连接节点实现游戏逻辑 |
| 蓝图 | Blueprint | 可视化编程的逻辑图，由节点和连线组成 |
| 节点 | Node | 可视化编程中的最小逻辑单元，如条件判断、循环、数学运算等 |
| 地编 / 地图编辑器 | Map Editor | 用于编辑地形、放置物件、设计场景的核心工具 |
| 载具编辑器 | Vehicle Editor | 自定义载具外观、属性和行为的编辑工具 |
| 材质编辑器 | Material Editor | 编辑物体表面材质属性（颜色、光泽、透明度等）的工具 |
| 粒子编辑器 | Particle Editor | 创建和调整粒子特效（火焰、烟雾、光效等）的工具 |

## 二、场景与物件

| 术语 | 英文 | 说明 |
|------|------|------|
| 互动物体 | Interactive Object | 场景中可被玩家交互的物件，如开关、按钮、宝箱、机关等 |
| 预制件 | Prefab | 由多个物件组合而成的可复用场景单元 |
| 虚拟投影装置 | Virtual Projector | 可在场景中播放动画或特效的投影设备 |
| 触发区域 | Trigger Zone | 检测玩家进入/离开的不可见区域，常用于触发事件 |
| 出生点 | Spawn Point | 玩家或 AI 的初始出生位置 |
| 路径点 | Waypoint | AI 巡逻或移动路线上的定位点 |

## 三、AI 与 NPC

| 术语 | 英文 | 说明 |
|------|------|------|
| 武装 AI | Armed AI | WoW 中的可编程 NPC/敌人实体，可配置行为和战斗逻辑 |
| AI 行为树 | AI Behavior Tree | 控制武装 AI 行为决策的树状逻辑结构 |
| 巡逻路径 | Patrol Path | 武装 AI 自动巡逻的预设路线 |
| 仇恨机制 | Aggro System | AI 识别和追踪目标的优先级系统 |

## 四、可视化编程概念

| 术语 | 英文 | 说明 |
|------|------|------|
| 全局变量 | Global Variable | 跨蓝图共享的数据变量，可在任意脚本中读写 |
| 局部变量 | Local Variable | 仅在当前蓝图中有效的数据变量 |
| 事件系统 | Event System | 基于触发器的事件响应机制，包括事件发送、监听和回调 |
| 事件广播 | Event Broadcast | 向所有或指定监听者发送自定义事件消息 |
| 条件节点 | Condition Node | 用于判断逻辑分支的节点（if/else） |
| 循环节点 | Loop Node | 用于重复执行操作的节点（for/while） |
| 函数封装 | Function Encapsulation | 将多个节点打包为可复用的自定义函数节点 |
| 数组操作 | Array Operation | 对列表数据的增删改查操作节点 |
| 定时器 | Timer | 延迟执行或周期性执行逻辑的时间控制节点 |

## 五、UI 与界面

| 术语 | 英文 | 说明 |
|------|------|------|
| 自定义 UI | Custom UI | 创作者自由设计的游戏内界面元素（按钮、文本、图片等） |
| HUD | Heads-Up Display | 游戏内常驻显示的状态信息界面（血量、分数等） |
| 锚点 | Anchor | UI 元素在不同分辨率下的定位参考点 |
| 适配 | Adaptation | UI 在不同设备屏幕尺寸下的自动调整机制 |

## 六、玩法系统

| 术语 | 英文 | 说明 |
|------|------|------|
| 组队系统 | Team System | 管理玩家队伍分组、匹配和协作的系统 |
| 计分系统 | Scoring System | 追踪和计算玩家/队伍得分的系统 |
| 排行榜 | Leaderboard | 展示玩家分数排名的功能 |
| 回合机制 | Round System | 控制游戏局数和回合切换的逻辑 |
| 重生机制 | Respawn System | 玩家死亡后重新出生的规则和配置 |
| 资源商城 | Resource Store | 创作者上传和下载资源（模型、音效、特效等）的平台 |

## 七、多媒体 & 资源

| 术语 | 英文 | 说明 |
|------|------|------|
| 骨骼动画 | Skeletal Animation | 基于骨骼绑定驱动的角色/物体动画 |
| 序列帧动画 | Sprite Animation | 逐帧播放的 2D 图片动画 |
| 音效系统 | Sound System | 管理背景音乐、环境音和交互音效的系统 |
| 天气系统 | Weather System | 控制场景中天气效果（雨、雪、雾等）的系统 |

## 八、发布与分享

| 术语 | 英文 | 说明 |
|------|------|------|
| 作品发布 | Publish | 将编辑完成的地图/玩法上传到服务器供玩家游玩 |
| 作品大厅 | Workshop / Lobby | 展示和浏览所有已发布作品的入口界面 |
| 模板 | Template | 官方或社区提供的预制玩法框架，可基于此快速创建新作品 |
| 审核 | Review | 作品发布前的内容合规性检查流程 |
