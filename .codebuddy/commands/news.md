# /news [关注方向]

## 功能
查看今日最新资讯推送，支持按关注方向筛选和推荐

## 参数
- `关注方向`（可选）：关注的重点领域，如 "AI"、"游戏"、"AI+游戏" 等。不指定则展示全部

## 执行步骤

1. **抓取资讯**：调用 `fetch_daily_digest` MCP 工具获取今日 RSS 订阅的最新文章列表
2. **分类整理**：将文章按领域分类（AI、游戏、技术、其他）
3. **重点推荐**：
   - 如果用户指定了关注方向，优先高亮该方向的文章
   - 特别关注 AI×游戏 交叉领域的内容
   - 为每篇文章标注推荐理由（一句话）
4. **展示列表**：以编号列表展示，标注分类和推荐星级（⭐~⭐⭐⭐）
5. **等待选择**：询问用户要保存哪些文章（输入编号），或是否需要抓取更多

## 输出格式示例

```
📰 今日资讯（2026-03-04）共 N 篇

🤖 AI 方向：
⭐⭐⭐ #1 《文章标题》 - 推荐理由
⭐⭐  #2 《文章标题》 - 推荐理由

🎮 游戏方向：
⭐⭐⭐ #3 《文章标题》 - 推荐理由

📖 其他：
⭐    #4 《文章标题》 - 推荐理由

💡 推荐阅读：#1, #3（与你的 AI+游戏 方向最相关）

输入文章编号保存到知识库（如 1,3,5），或输入 "more" 加载更多
```
