# /save <文章编号>

## 功能
保存 /news 推送中选定的文章到知识库，并自动抓取全文、翻译为中文

## 参数
- `文章编号`（必填）：要保存的文章编号，支持多种格式：
  - 单个：`1`
  - 多个：`1,3,5`
  - 全部：`all`
  - 跳过：`skip`

## 执行步骤

1. **保存选择**：调用 `save_selected_articles` MCP 工具，传入用户选择的编号
2. **逐一抓取**：对每篇保存的文章，使用 `collect_content` 抓取完整网页内容
3. **翻译处理**：如果原文是英文，翻译为中文，保留关键术语的中英对照
4. **结构化摘要**：为每篇文章添加：
   - 一句话总结
   - 3-5 个核心观点
   - 与用户工作的关联分析
5. **覆盖保存**：将翻译后的内容覆盖写入对应文件
6. **汇总报告**：展示处理结果

## 输出格式示例

```
📥 保存完成！共处理 3 篇文章：

✅ #1 《文章标题》→ ai/文件名.md
✅ #3 《文章标题》→ reading/文件名.md
⚠️ #5 《文章标题》→ 抓取失败（原因）

📌 核心收获：
- 文章1要点...
- 文章3要点...
```
