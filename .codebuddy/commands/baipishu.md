# /baipishu <关键词列表> [--version <版本号>] [--category <分类>]

## 功能
将简短的功能关键词扩写为标准、专业的 PUBGM WoW 版本白皮书功能点陈述句。支持单个或批量关键词输入，可按版本号或分类组织输出。

## 参数
- `关键词列表`（必填）：一个或多个功能关键词，多个关键词用中文/英文逗号分隔
- `--version <版本号>`（可选）：指定版本号，输出时在头部标注版本信息
- `--category <分类>`（可选）：指定功能分类，输出时按分类标题组织

## 执行步骤

1. **加载 Skill**：调用 `use_skill` 加载 `wow-whitepaper` Skill 获取扩写规则和术语参考
2. **解析输入**：识别关键词列表和可选参数（版本号、分类）
3. **逐条扩写**：对每个关键词执行扩写思维链（识别功能本体 → 推导操作机制 → 关联应用场景 → 套用句式）
4. **格式化输出**：根据是否有 --version 或 --category 参数，选择对应的输出格式
5. **用户确认**：展示扩写结果，等待用户确认或修改

## 使用示例

```
/baipishu 动画生成                              → 扩写单个功能点
/baipishu 动画生成, 自定义UI, 全局变量           → 批量扩写多个功能点
/baipishu --version 4.4.0 动画生成, 载具自定义   → 指定版本号扩写
/baipishu --category 可视化编程 循环节点, 事件广播 → 按分类扩写
```

## 输出格式示例

### 单条/批量模式
```
1. 新增动画生成功能，支持作者上传视频后生成对应骨骼动画，用于可视化编程控制武装AI和虚拟投影装置。
2. 新增自定义UI编辑器，支持创作者自由拖拽按钮与图片布局，用于制作个性化的游戏界面与交互菜单。
3. 新增全局变量管理面板，支持跨蓝图的变量声明与读写操作，用于多模块间的数据共享和状态同步。
```

```markdown
2. 新增载具自定义编辑器，支持调整载具速度、转向和碰撞属性参数，用于创作者打造个性化载具玩法。
```
