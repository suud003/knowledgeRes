# /generate_daily [YYYY-MM-DD]

## 功能
生成日报。自动读取当天（或指定日期）的打字数据，解析还原后智能识别工作内容，按项目分类整理，生成格式化日报并发布到 iWiki。

## 参数
- `YYYY-MM-DD`（可选）：指定日期，不传则默认使用当天日期（从系统时间获取）

## 执行步骤

1. **加载 Skill**：调用 `use_skill` 加载 `daily-weekly-report` Skill 获取完整工作流指引
2. **确定目标日期**：无参数则使用当天日期，有参数则使用指定日期
3. **读取打字数据**：从 `raw data/type_record/` 目录读取 `keyboard_{日期}.txt` 文件
4. **加载解析器**：调用 `use_skill` 加载 `type-record-parser` Skill
5. **解析打字数据**：按照 type-record-parser 规则将原始键盘记录还原为可读中文
6. **智能识别 + 分类**：从还原文字中识别工作内容并按项目分类
7. **生成日报**：使用日报模板生成格式化文档
8. **用户预览确认**：展示日报预览，等待用户确认或修改
9. **发布到 iWiki**：确认后发布到 iWiki 日报文件夹

## 使用示例

```
/generate_daily              → 生成今天的日报
/generate_daily 2026-03-04   → 生成2026年3月4日的日报
```

## 输出格式示例

```
📋 日报 - 2026年3月5日

## 今日工作

### 【项目A】
- [完成] 具体工作内容描述
- [进行中] 具体工作内容描述

### 【项目B】
- [进行中] 具体工作内容描述

## 问题与风险
- 问题描述

## 明日计划
- 计划内容
```
