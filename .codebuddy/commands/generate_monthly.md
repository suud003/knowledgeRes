# /generate_monthly [YYYY-MM]

## 功能
生成月报。自动读取本月（或指定月份）的所有打字数据，逐日解析后按周分组汇总，深度分析各项目进展，生成格式化月报并发布到 iWiki。

## 参数
- `YYYY-MM`（可选）：指定年月，如 `2026-02`
- 不传参数则默认生成本月的月报

## 执行步骤

1. **加载 Skill**：调用 `use_skill` 加载 `daily-weekly-report` Skill 获取完整工作流指引
2. **确定日期范围**：
   - 无参数：本月1日到今天
   - `YYYY-MM`：该月1日到该月最后一天
3. **批量读取打字数据**：从 `raw data/type_record/` 目录读取该月所有 `keyboard_{日期}.txt` 文件
4. **加载解析器**：调用 `use_skill` 加载 `type-record-parser` Skill
5. **逐日解析**：对每个文件按照 type-record-parser 规则还原为可读中文
6. **按周分组汇总**：将一个月的工作内容按自然周分组，深度分析各项目进展
7. **生成月报**：使用月报模板生成格式化文档（含按周进度、关键成果、问题风险等）
8. **用户预览确认**：展示月报预览，等待用户确认或修改
9. **发布到 iWiki**：确认后发布到 iWiki 日报文件夹

## 使用示例

```
/generate_monthly           → 生成本月的月报
/generate_monthly 2026-02   → 生成2026年2月的月报
```
