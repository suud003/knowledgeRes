# /digest

## 功能
生成本周知识库的学习摘要，帮助回顾和整理本周收获

## 参数
无

## 执行步骤

1. **获取本周内容**：调用 `search_recent`（days=7）获取本周所有新增内容
2. **分析整理**：按主题分类统计，提取关键洞察
3. **生成周报**：生成一份结构化的周学习摘要，包含：
   - 📊 本周数据：各主题新增数量
   - 🔑 关键洞察：本周最重要的 3-5 个收获
   - 🔗 知识关联：不同主题间发现的联系
   - 📌 下周建议：基于本周内容推荐的后续学习方向
4. **保存摘要**：将周报作为笔记保存到 reflection 主题

## 输出格式示例

```
📊 本周学习摘要（2026-02-26 ~ 2026-03-04）

📈 本周数据：共新增 15 条内容
   🤖 AI×6  🎮 游戏×4  💼 工作×3  📖 阅读×2

🔑 关键洞察：
1. Harness 工程是提升 Agent 性能的关键，核心在于上下文工程和自我验证
2. Agent 框架在演进：从链式调用 → 编排 → 自主 harness
3. 游戏设计学习的关键是"分而治之"——拆解操控手感、叙事、关卡等独立模块

🔗 知识关联：
- AI Agent 的 harness 设计思路 ↔ 游戏 AI 系统的行为树设计有异曲同工之处
- Agent 的自我验证循环 ↔ 游戏开发中的 playtesting 反馈循环

📌 下周建议：
- 深入研究 Agent 在游戏场景中的应用案例
- 尝试用 deepagents 框架搭建一个简单的游戏 AI demo
```
