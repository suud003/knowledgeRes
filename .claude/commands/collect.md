# /collect <url> [topic]

## 功能
AI 抓取指定网页正文 → 提取核心观点 → 生成结构化中文摘要 → 保存到知识库

## 参数
- `url`（必填）：文章链接
- `topic`（可选）：指定归类主题（如 ai、reading、product、work 等），不指定则根据内容自动判断

## 执行步骤

1. **抓取内容**：使用 `collect_content` MCP 工具抓取 url 的完整网页内容，topic 参数按用户指定传入（未指定则不传，让系统自动分类）
2. **读取原文**：读取抓取后保存的 markdown 文件内容
3. **翻译与摘要**：如果原文是英文，将其翻译为中文；提取核心观点，生成结构化摘要，格式如下：
   - **一句话总结**：用一句话概括文章核心
   - **核心观点**：3-5 个要点，每个用一句话概括
   - **关键概念**：文中提到的重要术语或概念
   - **与我的关联**：结合用户身份（游戏技术策划/AI Agent 研究者），分析这篇文章对用户可能的启发
4. **覆盖保存**：将翻译后的中文全文 + 结构化摘要覆盖写入原文件
5. **输出结果**：向用户展示一句话总结和核心观点

## 输出格式示例

```
✅ 已收集并处理：《文章标题》

📌 一句话总结：xxxxx

💡 核心观点：
1. xxxxx
2. xxxxx
3. xxxxx

🏷️ 已保存至：[主题名] / 文件路径
```
