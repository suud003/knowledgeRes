# /topics

## 功能
查看知识库的所有可用主题及其内容统计

## 参数
无

## 执行步骤

1. **获取主题列表**：调用 `list_topics` MCP 工具
2. **获取详情**：调用 `manage_topic`（action="list"）获取更详细的主题信息
3. **展示结果**：以表格形式展示所有主题，包含名称、描述、关键词

## 输出格式示例

```
📚 知识库主题一览

| 主题 | 描述 | 关键词 |
|------|------|--------|
| 🤖 ai | AI 相关 | AI, 大模型, Agent, prompt |
| 🎮 reading | 阅读学习 | 游戏设计, 阅读, 知识体系 |
| 💼 work | 工作记录 | 工作日记, 复盘 |
| 💡 ideas | 灵感想法 | 创意, 金句 |
| ... | ... | ... |

💡 提示：使用 /note <内容> <主题> 可以向指定主题添加笔记
```
