# Skill Creator

> 此命令为 Claude Code 适配版，实际 Skill 定义复用自 `.codebuddy/skills/` 目录。

## 概述
Skill 创建指南，提供输出模式、工作流参考和初始化/打包/验证脚本。

## 使用方式
执行此命令时，请依次读取以下文件获取完整指引：

### 核心定义
- `.codebuddy/skills/skill-creator/SKILL.md`

### 参考资料
- `.codebuddy/skills/skill-creator/references/output-patterns.md`
- `.codebuddy/skills/skill-creator/references/workflows.md`

### 脚本
- `.codebuddy/skills/skill-creator/scripts/init_skill.py`
- `.codebuddy/skills/skill-creator/scripts/package_skill.py`
- `.codebuddy/skills/skill-creator/scripts/quick_validate.py`
