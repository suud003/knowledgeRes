# /search <关键词> [topic]

## 功能
在知识库中搜索相关内容，支持跨主题或限定主题搜索

## 参数
- `关键词`（必填）：搜索关键词或问题
- `topic`（可选）：限定搜索的主题范围，不指定则搜索全部主题

## 执行步骤

1. **执行搜索**：调用 `query_context` MCP 工具，传入 query 和 topics（如指定）
2. **整理结果**：将搜索结果按相关度和时间排序
3. **展示结果**：以结构化方式展示匹配的记录，包含来源主题、时间和内容摘要
4. **如果结果不足**：尝试使用 `search_recent` 工具补充搜索最近的内容

## 输出格式示例

```
🔍 搜索「AI Agent」共找到 5 条结果：

📄 [AI 相关] 2026-03-04
   通过 Harness 工程提升深度 Agent 性能...

📄 [AI 相关] 2026-03-04
   关于 Agent 框架与 Agent 可观测性...

📄 [工作记录] 2026-03-01
   WOW AI 助手设计方案讨论...
```
