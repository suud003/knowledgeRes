# /recent [days] [topic]

## 功能
查看最近 N 天的知识库更新内容，快速回顾近期积累

## 参数
- `days`（可选）：查看最近多少天的内容，默认 7 天
- `topic`（可选）：限定查看的主题

## 执行步骤

1. **查询内容**：调用 `search_recent` MCP 工具，传入 days 和 topic
2. **按日期分组**：将结果按日期分组展示
3. **统计概览**：展示各主题的内容数量分布

## 输出格式示例

```

📄 [AI] 关于 Agent 框架与 Agent 可观测性
📄 [游戏] 学习游戏设计的困境

── 2026-03-03 ──
📄 [工作] WOW AI 助手方案设计...
```
